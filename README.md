Alcohol Consumption and Cancer Incidence Analysis

Objective  
Investigate whether national alcohol consumption correlates with cancer incidence rates.

Dataset  
- Alcohol consumption data (Our World in Data, 1990–2023)
- Global cancer incidence data (WHO GLOBOCAN)

Method  
- Data cleaning with pandas
- Country-level aggregation
- Correlation analysis
- Linear regression

Tools  
Python, pandas, matplotlib, seaborn, scipy

Result  
Correlation coefficient between alcohol consumption and cancer ASR: 0.42  
Regression R²: 0.18

Conclusion  
Moderate positive correlation observed at the country level, but causation cannot be inferred due to ecological fallacy.
